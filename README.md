#  Enhancing Digital Government and Economy(EDGE)

This repository contains a collection of projects developed during my **Python Training Program** conducted under the **EDGE (Enhancing Digital Government and Economy)** project of the **Bangladesh Computer Council (BCC)**, ICT Division.  

During this training, I learned the fundamentals of Python programming, data handling, and machine learning concepts.  
Each project in this repository reflects the skills and knowledge gained through practical exercises and guided learning sessions.

---

## üè¢ About EDGE Project

**Enhancing Digital Government and Economy (EDGE)** is a project of the **Bangladesh Computer Council (BCC)** under the **ICT Division**, jointly funded by the **World Bank** and the **Government of Bangladesh (GoB)**.  

The EDGE project aims to establish an **integrated, cloud-computing digital platform** for all government agencies and strengthen **cyber-security** infrastructure across the public sector.  
It is expected to save around **$200 million** in IT investments and enhance the government's capacity to deliver **virtual public services** efficiently during future crises.  

This training program was designed to enhance digital skills and promote the development of a skilled IT workforce for Bangladesh‚Äôs growing digital economy.

## üöÄ Projects Overview

### 1. **Car Recommendation System**
A recommendation system that suggests cars based on user preferences such as budget, type, and features.  
**Skills Used:** Python, Pandas, Scikit-learn, Machine Learning (Content-Based Filtering)

---

### 2. **Hotel Management System**
A simple management system to handle hotel bookings, check-ins, check-outs, and billing using Python.  
**Skills Used:** Python (OOP, File Handling), Tkinter

---

### 3. **Restaurant Management System**
A Python-based project to manage restaurant orders, menus, and billing through a user-friendly interface.  
**Skills Used:** Python, Tkinter, File Handling

---

### 4. **Movie Recommendation System**
A system that recommends movies based on user ratings and genres using data analysis and machine learning.  
**Skills Used:** Python, Pandas, Scikit-learn, Cosine Similarity

---

### 5. **Sentiment Analysis**
A Natural Language Processing (NLP) project that analyzes text data (e.g., reviews or tweets) to determine sentiment (positive, negative, neutral).  
**Skills Used:** Python, NLTK / TextBlob, Data Cleaning, Visualization

---

### 6. **Weather Prediction**
A project that predicts weather conditions such as temperature or rainfall using machine learning models.  
**Skills Used:** Python, Pandas, Scikit-learn, Linear Regression

---

### 7. **Number Recognition (Digit Classifier)**
A machine learning project that recognizes handwritten digits from image data (similar to the MNIST dataset).  
**Skills Used:** Python, TensorFlow / Keras, Neural Networks, Image Processing
